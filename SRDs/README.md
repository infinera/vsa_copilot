# SRDs

System Requirements Document (SRD) guidance

## Template

- **SRD ID:** SRD-<FeatureCode>-NNN
- **Related PRD ID:** PRD-<FeatureCode>-NNN
- **Functional Requirements:** list FR-### with details
- **Non-Functional Requirements:** NFR-### with measurable targets
- **Data Models:** pointers to `APIs/` and `Architecture/` artifacts
- **Traceability:** map SRD items to PRD FR-### and TestCase TC-###
